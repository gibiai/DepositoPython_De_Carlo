Python – Lezione 1 Fondamenti

Un buon programmatore deve:
	1.	Saper leggere il codice
	2.	Avere metodo e organizzazione
	3.	Saper scrivere codice chiaro e funzionante

Il codice deve essere leggibile, ordinato e replicabile

Cos’è Python

Python è un linguaggio di programmazione:
	•	Interpretato
	•	Ad alto livello
	•	Multi-paradigma (OOP, imperativo, funzionale)
	•	Dinamicamente e fortemente tipizzato

Python non viene eseguito direttamente dalla macchina.

Processo reale:
	1.	Il codice viene scritto
	2.	Viene trasformato in bytecode
	3.	Il bytecode viene eseguito dalla Python Virtual Machine (interprete)

Questo lo rende:
	•	molto flessibile
	•	più lento rispetto ai linguaggi compilati

Tipizzazione

Python è:
	•	Dinamico → il tipo di una variabile può cambiare
	•	Forte → ogni variabile ha sempre un tipo

Esempio:

x = 5        # intero
x = "ciao"   # stringa

Il tipo cambia solo con una nuova assegnazione (non automaticamente)

Paradigma OOP (Object-Oriented Programming)

Python supporta la programmazione orientata agli oggetti.
Principi fondamentali:
	•	Incapsulamento → nascondere i dettagli interni
	•	Ereditarietà → una classe può derivare da un’altra
	•	Polimorfismo → stesso metodo, comportamenti diversi

Concetto chiave: Astrazione = mostrare solo ciò che serve, nascondendo la complessità

Principali benefici:

Manutenibilità: Il codice è più facile da modificare grazie alla separazione delle responsabilità.

Riutilizzabilità: L'ereditarietà riduce la duplicazione del codice.

Sicurezza: L'incapsulamento previene modifiche accidentali ai dati.

Caratteristiche principali di Python
	•	Sintassi semplice e leggibile
	•	Molto flessibile (più modi per fare la stessa cosa)
	•	Alto livello di astrazione
	•	Supporta più stili di programmazione

 Ambiti di utilizzo

Python è molto usato in:
	•	Analisi dei dati
	•	Intelligenza artificiale
	•	Sviluppo software
	•	Automazione

 Sintassi base

Indentazione (fondamentale)
Python usa l’indentazione per definire i blocchi di codice.
Non usa {} o ;
Esempio:
if True:
    print("ciao")

Variabili

Servono per memorizzare valori:

nome = "Gabriel"
eta = 25

Operatori
	•	= → assegnazione
	•	== → confronto

Funzioni

Le funzioni sono blocchi di codice riutilizzabili.

Esempio:

print("ciao")

    •	print() è una funzione incorporata (built-in)

Le funzioni:
	•	possono essere richiamate
	•	possono usare variabili
	•	servono per organizzare il codice

Concetto chiave finale

 Scrivere codice significa creare istruzioni:
	•	chiare
	•	riutilizzabili
	•	prevedibili

Un buon codice funziona
Un ottimo codice si capisce
